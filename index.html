I'm a Postdoctoral Researcher in the Department of Computer Science and Engineering, School of Electronic Information and Electrical Engineering (SEIEE) at Shanghai Jiao Tong University (SJTU). Now I'm working in Sustainable Architectures and Infrastructure Laboratory ( SAIL)， as a part of Emerging Parallel Computing Center (EPCC). My PHD avisor includes Professor Chao Li, Professor Minyi Guo, and now my co-advisor is Professor Xiaokang Yang. I received my B.Eng. degree in Software Engineering from Northwestern Polytechnical University (NWPU) in 2018.

I graduated from SJTU in Dec. 2023 and got my PH.D. degree in Mar 2024.

I am a system person. I major in computer architecture and operating system. My research area includes disaggregated memory, graph processing, resource scheduling, heterogeneous architecture, etc.

The detailed research interests are listed as follows:

  • Hardware Architecture
    • Composable disaggregated architecture (CDI)
    • Disaggregated memory, disaggregated storage
    • Direct-connect components, like PCIe, RDMA, CXL, Smart NIC, etc.
    • Processing in/near memory
    • GPU Accelerator
    • FPGA Accelerator
  • Operating System
    • Far memory
    • Buffer and cache
    • Memory hierarchy, like page fault handling, swap, coherence, etc.
    • DGX, like GPU Direct Storage (GDS), GPUDirect RDMA, etc.
    • Resource control, such as Cgroup, zone, etc.
  • Runtime
    • Virtulization, management of VM, container, etc.
    • Resource scheduling
    • Data Profiling
    • High-Level Synthesis (HLS) Compilier
  • Big Data Application
    • Graph processing, like graph traversal, graph database, GNN, etc.
    • Netural networks, like DNN, Foundation Models, etc.
    • Cloud games
    • Database applications
    • Image detection and classifacation
